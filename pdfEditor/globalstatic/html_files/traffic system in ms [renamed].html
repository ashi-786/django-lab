Example of modeling and

simulating a traffic system

• This example models a simple road with vehicles following the Nagel-

Schreckenberg model, which is commonly used for simulating traffic

flow.

Nagel-Schreckenberg model

• The Nagel-Schreckenberg model is a cellular automaton model used

to simulate traffic flow on highways. It was developed by Kai Nagel

and Michael Schreckenberg in 1992 and is widely used because of its

simplicity and ability to reproduce real-world traffic phenomena like

traffic jams and stop-and-go waves.

• Step 1: Define Parameters

• The road is modeled as a 1D grid.

• Vehicles follow simple acceleration, deceleration, and random

slowing rules.

• The simulation runs for a given number of time steps.

clc; clear; close all;

% Parameters

road_length = 100; % Length of the road (cells)

num_vehicles = 20; % Number of vehicles

max_speed = 5; % Maximum speed of vehicles

num_steps = 50; % Number of simulation time steps

p_slowdown = 0.3; % Probability of random slowdown

% Initialize road with -1 (empty cells)

road = -ones(1, road_length);

% Place vehicles randomly on the road

positions = randperm(road_length, num_vehicles);

speeds = randi([0, max_speed], 1, num_vehicles);

for i = 1:num_vehicles

road(positions(i)) = speeds(i);

end

% Simulation Loop

for t = 1:num_steps

new_road = -ones(1, road_length); % Temporary road update

for i = 1:num_vehicles

% Get current position and speed

pos = positions(i);

speed = road(pos);

% Determine distance to next car

next_car = find(road(mod(pos + 1:pos + road_length, road_length) + 1) ~= -1, 1);

if isempty(next_car)

distance = road_length;

else

distance = next_car;

end

% Step 1: Acceleration

speed = min(speed + 1, max_speed);

% Step 2: Slowing down if necessary

speed = min(speed, distance - 1);

% Step 3: Random slowdown

if rand < p_slowdown && speed > 0

speed = speed - 1;

end

% Step 4: Update position

new_pos = mod(pos + speed, road_length) + 1;

new_road(new_pos) = speed;

% Store new positions

positions(i) = new_pos;

end

% Update road

road = new_road;

% Visualization

clf;

plot(positions, ones(1, num_vehicles), 'ro', 'MarkerFaceColor', 'r', 'MarkerSize', 10);

axis([1 road_length 0 2]);

title(['Traffic Simulation - Step ', num2str(t)]);

xlabel('Road Position');

pause(0.1);

end

Output

• How It Works

• Initialization: The road is defined as a 1D array with random vehicle

placements.

• Traffic Rules:

• Vehicles accelerate if possible.

• They slow down if there's a car ahead.

• A random slowdown factor is applied to simulate driver behavior.

• The new positions are updated.

• Visualization: The vehicles move along the road in a simple

simulation.
