• Mass-Spring-Damper System: Explained Simply

• The mass-spring-damper system is a basic mechanical system used

in modeling and simulation. It represents how objects behave when

forces act on them. This system is used in engineering, physics, and

control systems to study motion, vibrations, and stability.

• What Is the Mass-Spring-Damper System?

• It consists of three main components:

1️Mass (m) → A solid object (e.g., a car body, a pendulum, or a robot

part).

2Spring (k) → A component that resists displacement and tries to

return the system to its original position.

3️Damper (c) → A component that resists motion by dissipating

energy (like shock absorbers in a car).

• Example

• A car's suspension system is a mass-spring-damper system.

• The car body is the mass.

• The springs absorb the road bumps (spring k).

• The shock absorbers reduce vibrations (damper c).

% Parameters

m = 1; % Mass (kg)

c = 2; % Damping coefficient (Ns/m)

k = 10; % Spring constant (N/m)

F = 1; % Applied force (N)

% Time span for simulation

tspan = [0 10]; % Simulate from t=0 to t=10 seconds

% Initial conditions: [x0, v0] (initial displacement and velocity)

initial_conditions = [0; 0];

% Solve the ODE using ode45

[t, state] = ode45(@(t, state) massSpringDamper(t, state, m, c, k, F), tspan, initial_conditions);

% Extract displacement (x) and velocity (v) from the solution

x = state(:, 1);

v = state(:, 2);

% Plot the results

figure;

plot(t, x, 'b', 'LineWidth', 1.5);

xlabel('Time (s)');

ylabel('Displacement (m)');

title('Mass-Spring-Damper System Response');

grid on;

% Define the ODE function

function dstate = massSpringDamper(~, state, m, c, k, F)

x = state(1); % Displacement

v = state(2); % Velocity

% Dynamics: dx/dt = v, dv/dt = (F - c*v - k*x)/m

dxdt = v;

dvdt = (F - c*v - k*x)/m;

dstate = [dxdt; dvdt];

end

• Explanation

• Modeling:

• The system is modeled as a second-order ordinary differential equation

(ODE).

• The ODE is converted to a system of first-order ODEs using state

variables xx (displacement) and vv (velocity).

Simulation:

• MATLAB’s ode45 solver numerically integrates the ODE over the specified

time span.

• The function massSpringDamper defines the system dynamics.

• Results:

• The displacement x(t)x(t) is plotted over time.

• The response shows how the system oscillates and settles due to damping.
